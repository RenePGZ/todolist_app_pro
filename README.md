# ✅ ToDo App Pro

Aplicación de gestión de tareas multiusuario con autenticación, panel de administración, y despliegue listo para producción.

## 🚀 Tecnologías usadas
- Python 3.13
- Flask
- PostgreSQL
- SQLAlchemy
- Flask-Login / Flask-WTF
- Docker (WIP)
- Bootstrap 5
